# チーム名　愉快な仲間たち
## プロダクト概要
- プロダクト名：　Smi:Dule
- コンセプト： スケジュール共有アプリ
- 説明 ：　旅行計画などで使えるスケジュール管理アプリ。スケジュールや写真を一緒に行く友達と共有することができる。
- 推しポイント：　ファイヤーベースで、データを保存しているので、データをみんなで共有をすることができること。
